# Superstore Sales Analysis

## Overview
Sales data analysis performed on 9,970 records using Microsoft Excel.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## What's Inside
- Data Cleaning (duplicates, blanks, formatting)
- Profit Margin % calculation
- 5 Pivot Table analyses
- 4 Interactive Charts
- Professional KPI Dashboard
- Business Insights with Impact Ratings

## Key Findings
- West & East regions drive 60% of total revenue ($1.09M)
- Technology is most profitable category (53% of total profit)
- Tables sub-category losing $10,997 — needs pricing review
- California alone contributes 19% of total revenue

## Dataset
Source: Superstore Sales Dataset (Kaggle)
